# のぞき梅 着物お申込みフォーム

フォーム送信 → Slack通知（着物専用チャンネル）→ 振込案内メール自動送信

---

## セットアップ手順

### 1. リポジトリをGitHubに作成

GitHubで新しいリポジトリを作成し、このフォルダのファイルをすべてアップロードします。

### 2. Resend でメール送信の設定

1. https://resend.com でアカウント作成（無料）
2. 「Domains」→「Add Domain」で独自ドメインを追加
3. 表示されるDNSレコード（TXTレコード）をレンタルサーバーのDNS設定に追加
4. 「API Keys」→「Create API Key」でキーを発行

### 3. Slack Webhook の設定

1. Slack の「アプリを管理」→「カスタムインテグレーション」→「Incoming Webhooks」
2. 着物申込み用の新しいチャンネルを作成（例：`#着物-申込`）
3. そのチャンネルに Webhook URL を作成

### 4. Vercel にデプロイ

1. https://vercel.com でこのリポジトリをインポート
2. 「Environment Variables」に以下を設定：

| 変数名 | 値 |
|--------|-----|
| `RESEND_API_KEY` | Resend で発行したAPIキー |
| `FROM_EMAIL` | 独自ドメインのメールアドレス（例: info@your-domain.com） |
| `SLACK_WEBHOOK_URL` | Slack の Webhook URL |

3. デプロイ完了！

## ローカルで動かす場合

```bash
# 依存関係インストール
npm install

# 環境変数ファイルをコピーして値を記入
cp .env.local.example .env.local

# 開発サーバー起動
npm run dev
```

http://localhost:3000 で確認できます。

## カスタマイズ箇所

- **振込先口座情報** → `lib/email.ts` の口座情報部分を変更
- **商品リスト・金額** → `lib/types.ts` の `PRODUCTS` 配列を変更
- **メール文面** → `lib/email.ts` を編集
- **Slack通知内容** → `lib/slack.ts` を編集

## Googleスプレッドシート連携

### 手順

1. **Googleスプレッドシートを新規作成**

2. **Apps Script を開く**
   スプレッドシート上部メニュー「拡張機能」→「Apps Script」

3. **Code.gs の内容を貼り付ける**
   `gas/Code.gs` の内容をそのままエディタに貼り付けて保存

4. **トリガーを設定する**
   左メニュー「トリガー」→「トリガーを追加」
   - 実行する関数：`onEdit`
   - イベントのソース：スプレッドシートから
   - イベントの種類：編集時

5. **デプロイする**
   右上「デプロイ」→「新しいデプロイ」
   - 種類：ウェブアプリ
   - 次のユーザーとして実行：自分
   - アクセスできるユーザー：**全員**
   → 「デプロイ」→ 発行されたURLをコピー

6. **Vercel の環境変数に追加**
   `GAS_WEBHOOK_URL` = コピーしたURL

### スプレッドシートのステータス

| ステータス | 色 | 意味 |
|-----------|-----|------|
| 未入金 | 🔴 赤 | 申込み直後の初期状態 |
| 入金済み | 🟡 黄 | 振込確認済み |
| 制作中 | 🔵 青 | 制作・準備中 |
| 発送済み | 🟢 緑 | 発送完了 |
| キャンセル | ⚫ グレー | キャンセル |

ステータス列のドロップダウンを変更するだけで色が自動で変わります。
